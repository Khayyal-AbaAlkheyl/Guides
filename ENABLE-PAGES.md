# Enable GitHub Pages (required once)

The site files are ready. GitHub returns 404 until you turn Pages on:

1. Open: https://github.com/Khayyal-AbaAlkheyl/Guides/settings/pages

2. Under **Build and deployment** → **Source**, choose **Deploy from a branch**

3. Set **Branch** = `gh-pages` and folder **`/ (root)`** → **Save**

4. Wait 2–3 minutes, then open:

   **https://khayyal-abaalkheyl.github.io/Guides/bosnia/?lang=ar**

---

## Try on your PC right now (no GitHub needed)

Double-click `run-local.ps1` or run:

```powershell
cd C:\Users\khayy\yasserandme\travel-guides-publish
```

Then open: **http://localhost:8766/bosnia/?lang=ar**
